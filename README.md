# NYT

A web-based app that displays the New York Times Best Sellers and connects users to the places to buy them.

## Technologies Used
- HTML5
- CSS3
- JS
- jQuery
- [NYT Books API](https://developer.nytimes.com/docs/books-product/1/overview/)
- Vercel (for deployment)
