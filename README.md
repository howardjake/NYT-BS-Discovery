# NYT

A web-based app that displays the NYT Bestsellers lists.

## Technologies Used
- HTML5
- CSS3
- JS
- jQuery
- [NYT Books API](https://developer.nytimes.com/docs/books-product/1/overview/)
- Github Pages (for deployment)
